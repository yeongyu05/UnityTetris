# UnityTetris

## UI
 - 필드(Playfield)
    - 가로 * 세로 / 10 * 20
    - 필드 위쪽에 숨겨진 필드 4줄
 - 홀드 박스
 - 스코어
 - 다음 블록 표시
## 블록
 - 테트리스 블록 7가지 
    - I, O, Z, S, J, L, T 
### 블록 생성 규칙
 - 7종의 블록이 한 묶음으로 나온다. (Random Number Generator)
 - ex) 1 ~ 7번째 까지 I, O, Z, S, J, L, T 가 무작위로 나온 후 또 8 ~ 14번째 까지 I, O, Z, S, J, L, T 가 무작위로 나온다.
## 기능
 - 홀드
 - 게임시작
 - 일시정지
 - 다시하기
## 조작
 - 왼쪽으로 움직이기 / ←
 - 오른쪽으로 움직이기 / →
 - 소프트 드롭 / ↓
 - 하드드롭 / 스페이스 바
 - 시계 방향 회전 / ↑
 - 홀드 / shift
